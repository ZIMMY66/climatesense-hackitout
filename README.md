# 🌍 ClimaSense - AI Climate Anomaly Detection Dashboard

An AI-powered dashboard that detects unusual climate patterns and anomalies from environmental datasets using Machine Learning.

---

## 🚀 Problem Statement

Climate datasets contain hidden anomalies that are difficult to identify manually. Detecting these anomalies early can help researchers and policymakers make better decisions.

---

## 💡 Solution

ClimaSense uses Machine Learning and interactive visualizations to identify abnormal climate patterns and provide actionable insights through an easy-to-use dashboard.

---

## ✨ Features

✅ Upload climate datasets

✅ Interactive data visualizations

✅ AI-powered anomaly detection

✅ Dashboard interface

✅ Real-time anomaly analysis
