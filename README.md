**# ClimateSense - AI Climate Anomaly Detection

## Problem
Climate datasets contain hidden anomalies that are difficult to detect manually.

## Solution
ClimateSense is an AI-powered dashboard that detects abnormal climate patterns using machine learning.

## Features
- Upload climate dataset
- Interactive data visualization
- AI anomaly detection
- Dashboard interface
